# مدير أجهزة MikroTik

تطبيق ويب مطور باستخدام **FastAPI** يهدف إلى إدارة أجهزة **MikroTik RouterOS** من خلال واجهة ويب سهلة الاستخدام، دون الحاجة إلى استخدام برنامج **WinBox**.

---

## نبذة عن المشروع

يوفر هذا المشروع لوحة تحكم تساعد مدير الشبكة على الاتصال بجهاز MikroTik وإدارة أهم إعداداته من خلال المتصفح، وذلك باستخدام واجهة برمجية (API) خاصة بنظام RouterOS.

يهدف المشروع إلى تبسيط عمليات إدارة الشبكات وتوفير واجهة حديثة وسريعة مبنية على إطار العمل **FastAPI**.

---

## المميزات

* تسجيل دخول للمسؤول.
* لوحة تحكم رئيسية (Dashboard).
* عرض وإدارة واجهات الشبكة (Interfaces).
* عرض وإدارة عناوين IP.
* إدارة مستخدمي الـ Hotspot.
* واجهة ويب بسيطة وسهلة الاستخدام.
* سرعة عالية في الأداء بفضل FastAPI.
* استخدام جلسات (Sessions) للمصادقة.

## متطلبات التشغيل

* Python 3.10 أو أحدث.
* جهاز MikroTik يعمل بنظام RouterOS.

## تثبيت المشروع

### 1. استنساخ المشروع

```bash
git clone https://github.com/your-username/Router-Manager.git

cd Router-Manager
```

### 2. إنشاء بيئة افتراضية

#### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

#### Linux / macOS

```bash
python3 -m venv venv

source venv/bin/activate
```

### 3. تثبيت المكتبات

```bash
pip install -r requirements.txt
```

---

## إعداد الاتصال بالراوتر

قم بتعديل ملف **config.py** وإضافة بيانات جهاز MikroTik.

مثال:

```python
HOST = "192.168.88.1"
USERNAME = "admin"
PASSWORD = "your-password"
PORT = 8728
```

> تأكد من تفعيل خدمة **API** داخل إعدادات جهاز MikroTik.

---

## تشغيل المشروع

```bash
uvicorn main:app --reload
```

بعد التشغيل افتح المتصفح وانتقل إلى:

```
http://127.0.0.1:8000
```
